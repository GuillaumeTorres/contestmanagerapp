### Installation

```
Dans le dossier : 
$ npm install -g cordova ionic
$ npm install
$ gulp
$ ionic platform add [ ios | android ]
$ ionic build [ ios | android ]
$ ionic [ emulate | run ] [ ios | android ]
```
### Lancer le serveur

```
$ ionic server
```
